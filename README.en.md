## About Streamlit

> [!IMPORTANT]
> Streamlit is a Python library for building application interfaces focused on data science and machine learning, greatly simplifying and abstracting interface development.
>
> Additionally, Streamlit offfers a hosting platform that allows deploying projects built with its own library.
> 
> Link to the [Streamlit Website](https://streamlit.io)

---

> [!TIP]
> Libraries and technologies used:
> - numpy: A comprehensive library for all types of calculations and mathematics with Python.
> - pandas: Library built on top of numpy for data manipulation and processing.
> - matplotlib: Library for data visualization with all kinds of charts.
> - scikit-learn: Python library for Machine Learning.
> - yfinance: Yahoo Finance library for retrieving financial data in CSV format.
> - streamlit: Library for easily creating simple interfaces with Python.
